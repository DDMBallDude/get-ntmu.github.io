The Windows NT Modding Utility (NTMU) is a program that allows users to quickly patch and replace files in order to theme their system. NTMU can be used to automate the modification of resources in files on the user's system, including operating system files. Files can also be outright replaced with different ones, and new files can be added to the right location on the user's system. The tool additionally provides the ability to import registry configurations, allowing theme packs to automate the modification of the registry. Pack authors can define options for the end user to change in order to configure the theme pack to their liking before applying the pack to their system.
